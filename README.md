# Kaleb's Smart Room Controller

## Overview
The purpose of this project is to increase automation within Blink's (my cat) living environment.

1. Turn on the air freshener when the cat is detected entering her litter box
1. Turn on one soft colored light when the cat is detected near her food/water bowls.
1. Detect the temperature of the cat's room and light up LED's on the board accordingly.

Things to be determined:

1. Do something with the other 2 hue lights


## Details

This project uses the following components:

* Phillips hue lights
* Breadboard
* Wires
* Teensy 3.2 microcontroller
